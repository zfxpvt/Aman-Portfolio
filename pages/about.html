Hello,

I'm Aman Ahammed K.O, an aspiring cybersecurity professional based in Kerala, India. Currently working as an entry-level SOC Analyst, I spend my days monitoring security alerts, triaging incidents, and helping keep systems safe.

My journey into cybersecurity started with a deep curiosity about how systems work and how they can be broken and defended. That curiosity turned into a career path, and I haven't looked back since.

I'm currently pursuing the Certified Ethical Hacker (CEH) certification through EC-Council, building hands-on skills in ethical hacking, penetration testing, and offensive security techniques because the best defenders think like attackers.

When I'm not monitoring dashboards or studying for CEH, I'm working on personal cybersecurity labs, exploring CTF challenges, and staying current with the latest threat intelligence.
